# 穿越者引擎 (Traveller Engine) - 分步开发计划

为确保项目的高可行性与低风险落地，我们将系统拆分为四个里程碑阶段，从基础的“AI 读懂小说”逐步进阶到“复杂的沉浸式全游玩体验”。

---

## 里程碑 1 (M1)：数据基建与知识萃取核心
**目标：让机器读懂并记住一本长篇小说，建立结构化档案。**
*预期耗时：3-4 周*

### 核心任务：
1. **基础服务端搭建：**
   * 初始化后端工程（推荐 Python + FastAPI，更利于对接 LLM 生态）。
   * 部署大模型接口资源与 Zep Server 实例。
2. **文本清洗与向量化管道（Pipeline）：**
   * 编写脚本清洗小说文本（如去除多余换行符、提取章节标题）。
   * 将切分后的文本（Chunk）注入 Zep 的 Document Collection 中。
3. **静态知识图谱提取：**
   * 激活 Zep 的 Knowledge Graph Extraction，生成小说的“初始世界观设定”（人物、地点、核心道具等及关系边）。
   * 进行一次人工数据清洗与校对（针对少量提取错误的内容）。
4. **档案库前端 MVP：**
   * 搭建基础 Vue/React 前端框架。
   * 开发一个简单的档案馆页面，利用可视化图表库（如 G6 / Echarts）渲染 Zep 吐出的人物关系与实体节点。

---

## 里程碑 2 (M2)：创作推演引擎与核心 AI 对话流
**目标：构建导演 AI (Director AI)，能够在双轨模式下与玩家做出合理的交互。**
*预期耗时：4-5 周*

### 核心任务：
1. **Zep 动态 Session 管理：**
   * 设计会话管理系统，将每一个独立玩家的游玩实例绑定到一个独立的 Zep Session，确保记忆隔离。
   * 实现“读取存档 / 分支创建”功能：能够从某个历史节点复制 Zep Session 状态开启新的时间线。
2. **双轨 Director Prompt 工程设计：**
   * 编写**沙盒模式 (A)** 的 Prompt：配置极高自由度权重，强调根据自然法则和初始图设定进行推演。
   * 编写**收束模式 (B)** 的 Prompt：在 Prompt 中预留“剧情路标（Waypoints）”插槽。
3. **叙事动作解析器：**
   * 后端针对玩家输入进行类型分析：区分对话与肢体动作（如解析带 `/act` 前缀的字符串）。
4. **后台接口跑通与终端验证：**
   * 不依赖复杂前端，通过 Postman 或简单的命令行终端，测试一场具体的战斗或对话（如：刺杀某个 NPC），验证 Zep 是否正确总结了剧情，NPC 反应是否符合逻辑。

---

## 里程碑 3 (M3)：互动游玩端与沉浸式用户体验
**目标：交付给普通用户的游玩界面，实现 AVG 或高交互感的故事引擎。**
*预期耗时：3-4 周*

### 核心任务：
1. **开局创角流程开发：**
   * 开发“世界选择 -> 模式选择 (沙盒/原著) -> 角色创建 (原创/扮演)” 的引导交互。
   * 将用户的选择转化为底层 Zep 系统的初始 Core Memory。
2. **沉浸式互动 UI 开发：**
   * 摒弃传统的“微信聊天”布局，参考《AI Dungeon》或类跑团交互面板。
   * 左侧/顶部为跑团叙事文本，右侧为当前角色的状态栏（物品、体力、当前同伴等）。
   * 输入框支持快捷动作按钮。
3. **剧情改写（大纲生成）面板：**
   * 针对创作者提供非对话形式的改写面板，支持输入一段宏观剧情，让系统成片生成对应的小说章节文本。

---

## 里程碑 4 (M4)：DM 后台、动态图谱与产品闭环
**目标：为项目引入进阶的管理和复杂世界演进能力。**
*预期耗时：4 周*

### 核心任务：
1. **世界状态覆写机制 (Dynamic Lore Overwrite)：**
   * 随着 Zep 累积的滚动摘要（Rolling Summaries）达到阈值，提取最新重大事件。
   * 让新事件动态覆盖或更新 M1 阶段建立的静态设定图表（如：张三的派系从“正派”变为“邪派”）。
2. **叙事节奏控制器 (Pacing Controller)：**
   * 为模式 B 研发进阶算法计算“剧情停滞度”。当玩家漫无目的连续闲聊 3 个回合以上，主动抛出外部危机（如遇到强盗、失火等）推动剧情发展。
3. **上帝视角 DM 查阅台：**
   * 给开发团队和管理员提供的高级管理后台。
   * 可视化展示大模型推演时的“隐藏思考过程”、“当前玩家在系统眼里的好感度矩阵”、“正在生效的 Zep 关键记忆片段”。（对调优 Prompt 极具价值）。
